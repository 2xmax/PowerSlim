PowerSlim by Konstantin Vlasenko

Fitnesse Slim server implementation in PowerShell. PowerSlim makes it possible to use PowerShell for acceptance testing

PS> Get-PowerSLlm

Dependencies

The PowerShell 2.0 should be installed. PowerShell 2.0 is integrated with Windows 7 and Windows Server 2008 R2 and is released for Windows XP with Service Pack 3, Windows Server 2003 with Service Pack 2 and Windows Vista with Service Pack 1.

Install

!define TEST_SYSTEM {slim} !define COMMAND_PATTERN (PowerShell -NonInteractive -ExecutionPolicy unrestricted -file .\slim.ps1)
